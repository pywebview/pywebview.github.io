快速入门

# 快速入门

# 安装:

pip install pywebview

在 Linux 上，您需要添加其他依赖库。详细信息请参阅安装页面。

# Hello world

import webview
webview.create_window('Hello world', 'https://pywebview.github.io/docs/hello')

进一步探索 pywebview，请查阅指南，示例或者贡献等页面。
